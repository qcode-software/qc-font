# qc-font
Qcode Fonts

Sample CSS to load qc-fonts
```css
/* Font */
@font-face {
    font-family: 'qc-font';
    src: url('https://d3nz4z14c0bzit.cloudfront.net/qc-font/?.?.?/qc-font.eot');
    src: url('https://d3nz4z14c0bzit.cloudfront.net/qc-font/?.?.?/qc-font.eot?#iefix') format('embedded-opentype'),
    url('https://d3nz4z14c0bzit.cloudfront.net/qc-font/?.?.?/qc-font.woff') format('woff'),
    url('https://d3nz4z14c0bzit.cloudfront.net/qc-font/?.?.?/qc-font.ttf') format('truetype'),
    url('https://d3nz4z14c0bzit.cloudfront.net/qc-font/?.?.?/qc-font.svg#qc-font') format('svg');
    font-weight: normal;
    font-style: normal;
}
```
